# LIGHTFAB

This is the website of the LIGHTFAB initiative, a thematic semester funded by the [Complex Systems Academy of Excellence](https://univ-cotedazur.fr/recherche-innovation/structures-de-recherche/academies-dexcellence/academie-2/a-propos/lacademie-systemes-complexes) at the [Université Côte d'Azur](https://univ-cotedazur.fr/) in the framework of the IdEx program.

This is built with [Astro](https://astro.build) from the [Accessible Astro Starter](https://accessible-astro.incluud.dev/themes/accessible-astro-starter/) theme.

## How to contribute (collaborators only)?

1. Clone this theme locally 

2. Run any of the following commands in your terminal:

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Installs dependencies                        |
| `npm run dev`     | Starts local dev server at `localhost:4321`  |
| `npm run build`   | Build your production site to `./dist/`      |
| `npm run preview` | Preview your build locally, before deploying |

3. Push your modifications to deploy
